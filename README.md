# Udemy Clone

A simple **Udemy Clone** built using **HTML and CSS** that recreates the look and feel of the Udemy homepage. This project was created to practice front-end development and improve skills in HTML, CSS, Flexbox, and responsive design.

## 🚀 Features

- Responsive navigation bar
- Search bar
- Course categories
- Recommended course section
- Clean and modern UI
- Mobile-friendly layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox

## 📂 Project Structure

```
Udemy-Clone/
│── index.html
│── style.css
│── images/
└── README.md
```

## 🎯 Purpose

The main goal of this project is to strengthen HTML and CSS fundamentals by recreating a popular e-learning website layout and improving responsive web design skills.
